# 🧺 ***Smart Pantry – Expiry Alerter***

Smart Pantry is a **full-stack web application** designed to solve a **real-world daily problem — food wastage due to forgotten expiry dates**.  
It helps users **manage their pantry efficiently**, **track grocery expiry dates**, and **make smarter cooking decisions** with the help of **AI-powered suggestions**.

The application allows users to log grocery items manually or via **barcode scanning**, automatically identifies products, and highlights items that are **about to expire**.  
Based on these items, Smart Pantry suggests **relevant recipes** and provides an **AI Cooking Assistant** that helps with recipe ideas, substitutions, and quick cooking tips.

***Smart Pantry focuses*** on:
- Practical usefulness  
- Clean full-stack architecture  
- Intelligent use of APIs and AI  
- Scalable and extensible design  

The goal is simple: **waste less food, save money, and cook smarter** 🍽️

---

***## 🚀 Key Features***

- 📝 Add grocery items manually with expiry dates
- ⏰ **Expiring Soon** alerts for items near expiry (Items that are expiring in 3 days or less then 3 days)
- 💡 Item recognition using OpenFoodFacts API
- 🧾 Barcode lookup using Google Vision AI
- 📷 Barcode scanning from images using Google Vision AI
- 🍳 Recipe suggestions based on available/expiring items
- 🤖 AI Cooking Assistant for:
  - Recipe ideas
  - Ingredient substitutions
  - Quick cooking tips
- 🌙 Clean UI with Dark Mode support
